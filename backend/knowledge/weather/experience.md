# Weather Analyst 经验文档

## 范围
判断气象灾害的**触发条件**（高温热害、大风、暴雨、干旱）。
本 Agent 只输出气象层面的判断，不结合作物 / 地块脆弱性 —— 那些由 Crop / Plot Analyst 负责。

## 关键经验（人话）

### 高温热害
- 35℃ 是常用的农业气象阈值，但**敏感生育期**（玉米抽雄、小麦灌浆）会降到 32-33℃
- 仅看单点温度容易误报，必须看**累计高温小时数**
- 干热风（高温 + 低湿 + 中等风）杀伤力远大于纯高温

### 大风 / 倒伏诱因
- 6 级（10.8-13.8 m/s）是玉米拔节期倒伏的高风险线
- 7 级（13.9-17.1 m/s）几乎必倒，与生育期无关
- 持续时间 >3h 比瞬时阵风更危险

### 暴雨
- 24h ≥ 50mm = 暴雨，≥ 100mm = 大暴雨
- 关键是看**作物所处生育期 + 排水条件**（这部分让 Plot Analyst 来加权）

### 干旱
- 0-10cm 土壤含水量 < 0.18（体积含水率）连续 3 天 + 24h 降水 < 1mm = 表层干旱
- 玉米抽雄-灌浆撞干旱 = "卡脖旱"
