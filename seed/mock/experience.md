# Decision Agent 经验库种子（experience.md）

> 用法：Decision Agent 在生成预警话术 / 给保险公司的描述时，把这里当成 RAG 语料。
> 8 条都是真实历史灾情的脱敏复述。演讲时如果被问"你这模型见过啥"，就把这页翻出来。
> 每条按统一 schema：场景 → 触发条件 → 损失链 → 后视应对。

---

## CASE-001 · 2023 河南"烂场雨" 小麦穗发芽

- **场景**：豫南、豫中冬小麦，2023-05-25 进入蜡熟期。
- **触发条件**：5/25–5/30 连续 6 天阴雨 + 日均湿度 > 85% + 日均温 22–26°C，部分县日降水 50mm 以上。
- **损失链**：成熟期穗上发芽 → 容重下降 → 收购等级降至饲料级 → 局部减产 30–50%。
- **后视应对**：T-3 天即应触发抢收预案、协调烘干塔、保险预核启动；T-1 天再动已无用，谷物已立穗发芽。
- **风险标签**：`peril=HEAVY_RAIN_AT_HARVEST`, `crop=winter_wheat`, `stage=蜡熟-收获`.

## CASE-002 · 2023 皖北 6 月持续高温 玉米抽雄花期受阻

- **场景**：安徽宿州、阜阳夏玉米，2023-06-22 进入抽雄期。
- **触发条件**：6/22–7/04 连续 13 天日最高 ≥ 38°C，夜温 ≥ 27°C，相对湿度 < 40%。
- **损失链**：花粉活力下降 → 雌雄不育 → 空粒、秃尖 → 单产损失 15–25%。
- **后视应对**：抽雄前 7 天即应触发"高温热害"预警 + 推荐喷灌降温/叶面磷酸二氢钾；保险走"高温热害附加险"。
- **风险标签**：`peril=HEAT_STRESS_AT_FLOWERING`, `crop=summer_maize`, `stage=抽雄-吐丝`.

## CASE-003 · 2024 黑龙江夏季大风 玉米连片倒伏

- **场景**：黑龙江绥化、齐齐哈尔春玉米，2024-08-08。
- **触发条件**：北上台风外围下沉 + 副高边缘 → 阵风 9–10 级，土壤含水率 35% 以上（土松根浅）。
- **损失链**：拔节后期-灌浆期茎倒伏 + 部分根倒伏 → 机械收获难度上升 → 倒伏地块单产 -20%，倒伏后霉变率升高。
- **后视应对**：T-48h 风预报已经能看见，应立即触发减灾建议（喷施矮壮素/抗倒剂为时已晚，但可提前协调人工扶禾设备）；保险查勘队 T-24h 进入待命。
- **风险标签**：`peril=WIND_LODGING`, `crop=spring_maize`, `stage=灌浆`.

## CASE-004 · 2022 长江中下游 7–8 月伏旱

- **场景**：湖北、湖南、安徽南部一季稻 + 中稻，2022-07-10 起。
- **触发条件**：连续 40 天日最高 ≥ 35°C + 累积降水 < 20mm + 库塘蓄水 < 60% 常年。
- **损失链**：抽穗扬花期高温干旱 → 结实率显著下降 → 部分田块绝收。
- **后视应对**：6 月下旬应已基于 ENSO + 副高强度做 60 天季节预报，T-30 天推送抗旱品种切换 / 提水灌溉路线已经晚；T-60 天才是黄金窗口。**穗安长期路线必须做季节尺度归因，不能只盯 7–15 天。**
- **风险标签**：`peril=DROUGHT_SEASONAL`, `crop=mid_rice`, `stage=抽穗-灌浆`.

## CASE-005 · 2024 山东烟台冰雹打果

- **场景**：胶东半岛苹果产区，2024-06-12。
- **触发条件**：午后强对流，0°C 层高度 4.5km，CAPE > 2500 J/kg，雷达回波 ≥ 55dBZ，冰雹直径 1.5–3cm。
- **损失链**：果实表面伤痕 → 商品果率从 85% 降至 40% → 部分果园改加工销售，单亩毛利 -3000 元。
- **后视应对**：冰雹预警窗口短（< 2h），靠预报抢不出来；演讲时强调**穗安在冰雹场景下的价值是 T-0 自动启动理赔流程，而不是预防**。
- **风险标签**：`peril=HAIL`, `crop=apple`, `stage=膨果期`.

## CASE-006 · 2018 黄淮"干热风" 小麦千粒重下降

- **场景**：山东、河南、河北南部冬小麦灌浆期，2018-05-17 至 05-22。
- **触发条件**：日最高 ≥ 32°C + 14 时湿度 ≤ 30% + 14 时风速 ≥ 3 m/s，连续 3 天即构成"重干热风"。
- **损失链**：灌浆受阻、叶片早衰 → 千粒重下降 2–4g → 单产 -8% 至 -15%。
- **后视应对**：T-5 天可基于温度/湿度/风的三因子滚动预报触发；推荐喷施磷酸二氢钾 + 增加灌水。
- **风险标签**：`peril=DRY_HOT_WIND`, `crop=winter_wheat`, `stage=灌浆`.

## CASE-007 · 2020 东北"台风三连击" 玉米大面积倒伏

- **场景**：黑龙江、吉林春玉米，2020-08-27 至 09-08。
- **触发条件**：台风"巴威""美莎克""海神"10 天内三次北上，玉米灌浆-乳熟期遭遇连续大风 + 强降水土壤饱和。
- **损失链**：倒伏面积 1100 万亩 + 部分穗位水浸霉变，省内玉米总产损失约 5%。
- **后视应对**：单次台风可预报 5–7 天；连续多次的累积效应需要 Harness 做"累积风险加权"，单次模型会低估第三次的边际损失。
- **风险标签**：`peril=WIND_LODGING_CUMULATIVE`, `crop=spring_maize`, `stage=灌浆-乳熟`.

## CASE-008 · 2023 江汉平原 5 月寡照阴雨 油菜菌核病

- **场景**：湖北荆州、潜江油菜，2023-05-01 起。
- **触发条件**：开花-结荚期连续 12 天阴雨 + 日照 < 2h/天 + 日均湿度 ≥ 85%。
- **损失链**：菌核病爆发，发病率 30–60%，菜籽含油率下降 2–3 个百分点。
- **后视应对**：花期前 7 天施菌核净 + 多菌灵是关键窗口；T-10 天的湿度预报已可触发预防性喷药建议。
- **风险标签**：`peril=SCLEROTINIA_HUMIDITY`, `crop=rapeseed`, `stage=花期-结荚`.
